# 🎓 CSN25 - Computação Simbólica e Numérica

> Repositório central de códigos e materiais da disciplina ministrada em 2025.

---

## 🚀 Sobre a Disciplina
Este repositório consolida o desenvolvimento prático de algoritmos voltados à inteligência computacional, com foco especial em **Visão Computacional** e **Eficiência de Processamento**.

## 🧠 Tópicos Principais
| Tópico | Descrição | Status |
| :--- | :--- | :--- |
| 🖼️ **CNNs** | Implementação de Redes Neurais Convolucionais. | 🟢 Concluído |
| 🎯 **YOLO** | Detecção de objetos em tempo real (v8/v11). | 🟢 Concluído |
| 👁️ **OpenCV** | Processamento de imagens e filtros espaciais. | 🟢 Concluído |
| ⚡ **Otimização** | Modelos para sistemas de tempo real (Edge AI). | 🟡 Em andamento |

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python 3.x
* **Frameworks:** PyTorch, TensorFlow
* **Bibliotecas:** OpenCV, Scikit-learn, Pandas, Matplotlib
* **Otimização:** ONNX, TensorRT (foco em hardware restrito)

## 📁 Estrutura do Repositório
* `/yolo`: Scripts de detecção e pesos dos modelos.
* `/opencv`: Filtros e algoritmos de processamento de imagem.
* `/datasets`: Amostras leves para testes de inferência.
