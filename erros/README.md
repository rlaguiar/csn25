# README - Disciplina de Computação Simbólica e Numérica (2025)

Este diretório contém os códigos relacionados à disciplina de Computação Simbólica e Numérica do ano de 2025.

## Objetivo

Organizar exemplos, exercícios e soluções relacionados a temas de computação simbólica e numérica, incluindo:
- Análise de erros numéricos
- Aritmética de ponto flutuante
- Estabilidade e condicionamento
- Métodos numéricos básicos

## Estrutura

Cada arquivo neste diretório corresponde a um exercício ou implementação específica da disciplina.

## Uso

1. Abra o arquivo desejado no ambiente de desenvolvimento.
2. Leia os comentários e as instruções no próprio código.
3. Execute os exemplos ou testes conforme descrito.
